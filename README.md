# Terminal RPG

A turn-based fantasy RPG built entirely in JavaScript and played through the terminal.

This project started as a way to learn JavaScript fundamentals by building something practical instead of following small coding exercises. Rather than creating isolated examples, every new programming concept is added directly into the game as a real feature.

## Features

* ⚔️ Turn-based combat
* 🗡️ Multiple weapons with unique damage values
* ✨ Mana and spell system
* ❤️ Health tracking with current and maximum HP
* 🧪 Potion system
* 👹 Random enemy encounters
* 📈 Experience and leveling system
* 🧭 Area exploration
* 🎲 Random events
* 📖 Story progression
* 🎨 Terminal UI with colors, ASCII art, and combat effects *(planned)*

## Built With

* JavaScript (Node.js)

## Running the Project

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/terminal-rpg.git
```

Navigate into the project:

```bash
cd terminal-rpg
```

Run the game:

```bash
node terminalRPG.js
```
